# EasyCalling
EasyCalling is the tool for termux user to make a call easily from termux. 
### Installation Guide
```
pkg update && pkg upgrade 
```
```
pkg install git
```
```
git clone https://github.com/MaverickSoftware/Easycalling

```
```
cd Easycalling
```
```
bash setup
```
### To Run
just type following command in terminal.
```
call
```
## To Uninstall
just type following command in terminal.
```
bash uninstall
```
